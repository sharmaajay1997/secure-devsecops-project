# 🚀 End-to-End DevSecOps Project (AWS EKS)

## 🔥 Overview
This project demonstrates a complete DevSecOps pipeline with CI/CD, security scanning, monitoring, and logging.

## ⚙️ Tech Stack
- GitHub
- Jenkins
- SonarQube
- Docker
- Trivy
- Kubernetes (EKS)
- Falco
- Prometheus + Grafana
- Elasticsearch + Kibana

## 🔐 Security
- Gitleaks (Secrets Scan)
- SonarQube (SAST)
- Trivy (Image Scan)
- Falco (Runtime Security)

## 📊 Monitoring & Logging
- Prometheus + Grafana dashboards
- Elasticsearch + Kibana for logs

## 🚀 Deployment Flow
GitHub → Jenkins → Docker → Trivy → EKS → Monitoring → Logging
